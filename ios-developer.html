IOS Developer: Contractual

Location: New Delhi

As an iOS Developer at Green Apple Solutions, you will be responsible for the development and architecture of our various mobile app projects. The role involves working closely with a talented development team, brainstorming UI/UX, and implementation in Objective-C.

Job Responsibilities:
  • Build native applications for iOS - driving the development of delightful user experiences for our customers
  • Deliver across the entire app life cycle –concept, design, build, deploy, test, release to app stores and support
  • Test, and maintain, against multiple hardware and software versions
  • Well-versed in with REST APIs and JSON
  • Collaborate with cross-functional teams to define, design & ship new features
  • Work on bug fixing and improving application performance
  • Unit-test code for robustness, including edge cases, usability, and general reliability
  • Build prototypes at tech scoping stage of projects
  • Working with the front end developers to build the interface with focus on usability features
  • Optimising performance for the apps
  • Keep up to date on the latest industry trends in the mobile technologies
Job Requirements:
  • 1 - 3 years of relevant experience
  • B.Tech/MCA from a good college/university preferred / A Bachelor's degree in Computer Science or related field
  • Strong understanding of interactive application development paradigm under iOS, memory management, file I/O, network programming, concurrency and multi-threading
  • Strong OO design and programming skills in Objective-C
  • Familiar with iOS SDK (UIKit, Cocoa Touch, Core Data, Core Location, etc)
  • familiar with xcode
  • Expertise in working on multi-threaded apps, XML & JSON, and web services
  • Exposure to Location-Based Services, Supporting multiple screens, Security and Permissions, Widgets and social APIs integration will be a plus
  • Coding standards, organizational programming skills and good commenting practices is important
  • Quick learner, team player, always up to date with new technologies
